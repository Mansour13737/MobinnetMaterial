# **App Name**: TowerTrack

## Core Features:

- Data Import: Import material data from Excel files into the system. Can import files up to 9000 records.
- Intelligent Search: AI-powered search functionality with typo detection, understanding of human descriptions (e.g., '20-meter light tower conversion' or 'galvanized 12 bolt'), and suggestion of similar results.
- Inventory Management: Track material inventory by recording incoming and outgoing items, managing the condition (e.g., healthy/defective), adding new items, and editing records.
- Reporting Dashboard: Management dashboard displaying inventory status, number of healthy/defective items, monthly consumption reports, and critical material alerts.
- Condition Tool: Use generative AI tool for suggestion of similarly categorized items to the existing items for creating reports based on name, type, code, category and status of items. 
- User Access Levels: Allow definition of user access levels (admin, warehouse manager, observer).
- Export: Enable export of the data to Excel and PDF

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a modern, techy feel.
- Complementary color: Metallic silver-gray (#C0C0C0).
- Accent color: Industrial yellow (#FFEA00) for emphasis and CTAs.
- Background: Modern white with subtle shadows (#FFFFFF).
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent, minimalist icons to represent inventory status and actions.
- Dashboard-focused design with a large, prominent search bar.
- Subtle transitions and animations for a smooth user experience.
